
# TypeCodeBackEnd
TypiCodeBackEnd
This is an ASP .Net Core Web API back-end to consume end points from https://jsonplaceholder.typicode.com/

- /posts/1/comments
- /albums/1/photos
- /users/1/albums
- /users/1/todos
- /users/1/posts




## Features


- Register log server using Serilog
